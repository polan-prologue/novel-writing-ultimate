## 角色层级规划

### 确定角色体系规模

| 规模 | 人数 | 角色构成 | 适用场景 |
|:----:|:----:|----------|----------|
| **微型** | ≤10人 | 主角1 + 女主1 + 反派1 + 配角2-3 + 路人若干 | 短篇/单线剧情 |
| **小型** | 15-25人 | 主角1 + 女主1 + 重要配角2-3 + 次要配角3-5 + 反派1-2 + 小反派2-3 + 路人 | 中篇/双线剧情 |
| **中型** | 30-50人 | 主角1 + 女主1 + 男二/女二1 + 重要配角3-5 + 次要配角5-8 + 反派1-2 + 阶段性对手3-5 + 势力代表2-3 + 路人 | 长篇/多线剧情 |
| **大型** | 50-100人 | 完整主角团 + 多女主/多男主 + 势力核心成员 + 各方势力代表 + 多个反派 + 配角群像 + 路人 | 超长篇/史诗剧情 |

### 核心三角（必做）

```
【核心三角】
① 主角（故事核心视角，必须首先确定）
   → 名字：____________
   → 类型：正派主角 / 反派主角 / 中立主角

② 女主/情感核心（可选项）
   → 名字：____________
   → 与主角关系：恋人 / 暧昧 / 冤家 / 师徒 / 亲情

③ 主反派/核心冲突（必须至少一个）
   → 名字：____________
   → 与主角关系：死敌 / 竞争 / 追杀 / 操控
```

| 等级 | 描述 |

## 阵营体系

### 阵营类型

| 阵营 | 特点 | 潜在问题 |
|------|------|----------|
| **正道阵营** | 名门正派/朝廷官府/秩序维护者；正义、秩序、规则 | 伪善、道貌岸然、食古不化 |
| **魔道阵营** | 邪魔外道/叛军/起义军/革命者；自由、反叛、打破规则 | 残忍、极端、无底线 |
| **中立阵营** | 逍遥散人/独立势力/雇佣兵；独立、不受约束、注重利益 | 摇摆不定、缺乏信念 |

## 角色弧线

### 角色弧线类型

| 序号 | 类型 | 说明 |
|:----:|------|------|
| 1 | 成长型弧线 | 废柴→强者：起点弱 → 经历磨难 → 变强 → 达成目标 |
| 2 | 堕落型弧线 | 正→邪：正直 → 遭遇不公 → 信念崩塌 → 黑化 |
| 3 | 救赎型弧线 | 邪→正：堕落 → 遇到关键人物 → 唤醒良知 → 自我救赎 |
| 4 | 认知型弧线 | 误解→理解：偏见 → 经历事件 → 改变认知 → 获得成长 |
| 5 | 悲剧型弧线 | 希望→毁灭：满怀希望 → 不断努力 → 命运无常 → 悲剧结局 |
| 6 | 觉醒型弧线 | 沉睡→觉醒：平凡/伪装 → 触发事件 → 真实身份/能力觉醒 → 新生 |
| 7 | 坚守型弧线 | 不变→坚守：坚守初心 → 历经考验 → 保持本心 → 获得认可 |
| 8 | 无弧线 | 静态角色：性格能力贯穿始终，作为参照系或功能性角色 |
