## 主线剧情设计

### 主线剧情类型

| 类型 | 描述 | 适用题材 |
|------|------|----------|
| **复仇线** | 主角遭受不公→隐忍蓄力→反击复仇 | 都市重生、玄幻废柴流 |
| **崛起线** | 废柴起步→层层突破→登顶巅峰 | 玄幻修仙、系统流 |
| **守护线** | 弱小起点→保护重要之人→成为支柱 | 都市、甜文 |
| **探寻线** | 寻找真相/身世/宝物→揭开谜底 | 悬疑、探险 |
| **统治线** | 建立势力→统一天下/行业 | 玄幻、种田、商战 |
| **救赎线** | 弥补遗憾/拯救他人→自我救赎 | 重生、穿书 |

## 支线剧情设计

### 支线剧情类型

| 类型 | 作用 | 占比建议 |
|------|------|----------|
| **感情支线** | 丰富爱情/友情/亲情线 | 15-20% |
| **人物背景支线** | 揭示配角/反派动机 | 10-15% |
| **势力支线** | 展示其他势力/门派故事 | 10-15% |
| **任务支线** | 独立小故事，获取资源/情报 | 10-15% |
| **回忆支线** | 补充世界观/角色过往 | 5-10% |
| **暗线支线** | 埋设伏笔，后期反转 | 5-10% |

```

名称：____________

核心内容：____________

【剧情结构】
起：第__章，契机是____________

转：第__章，危机/转折是____________
合：第__章，结局是____________

【与主线关系】
├─ 为主线提供：____________（资源/情报/情感支撑）
├─ 回收伏笔：____________

【独立性评估】
□ 高（可单独抽出阅读）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 支线剧情设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **可独立** | 支线应有头有尾，不依赖主线也能理解 | 配角个人故事完整 |
| **有意义** | 服务于人物塑造或世界观完善 | 揭示反派动机 |
| **有钩子** | 支线结尾连接主线，吸引继续阅读 | 任务中获得主线关键道具 |
| **有限度** | 占比合理，不喧宾夺主 | 单支线不超过3章 |
| **有呼应** | 与主线或其他支线形成呼应 | 同一事件不同视角 |

## 多线并发设计

### 多线并发模式

| 模式 | 描述 | 适用场景 |
|------|------|----------|
| **主副并行** | 主线为主，支线穿插 | 绝大多数网文 |
| **双线交织** | 两条线交叉叙述，最终汇合 | 复仇+感情 |
| **三线鼎立** | 三条主线交织，势力对决 | 玄幻多势力 |
| **群像多线** | 多角色多线并行，视角切换 | 悬疑、群像文 |
| **时空交错** | 现实线+回忆线交织 | 重生、穿越 |
| **平行世界** | 不同世界/位面同时推进 | 穿越、系统 |


### 多线切换技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **悬念切换** | A线关键时刻切换到B线 | 章末钩子应用 |
| **时间同步** | 同时空事件多视角呈现 | 决战前各方准备 |
| **因果串联** | A线事件影响B线发展 | 配角行动改变主线 |
| **伏笔呼应** | A线埋设，B线回收 | A线获得线索，B线使用 |
| **情绪调节** | 紧张主线后切换轻松支线 | 大战后的日常 |

## 剧情节奏模板

### 章节节奏模板

```
【单章节奏】
┌─────────────────────────────────────────────────────────────┐
│  【起】第1-2段：承接上文，引出新事件（200-500字）              │
│  【承】第3-5段：事件展开，细节铺陈（800-1500字）              │
│  【转】第5-6段：出现转折/危机/意外（300-500字）               │
│  【合】第7段：留下钩子，连接下一章（100-300字）               │
└─────────────────────────────────────────────────────────────┘

【卷/大章节节奏】
┌─────────────────────────────────────────────────────────────┐
│  卷首3章：世界观展开 + 核心人物登场 + 主线启动               │
│  卷中10章：剧情推进 + 支线穿插 + 小高潮2-3个                 │
│  卷末3章：大高潮 + 重大转折 + 悬念留白                       │
└─────────────────────────────────────────────────────────────┘
```
