## 文风分析维度

### 五大分析维度

| 维度 | 分析内容 | 示例 |
|------|----------|------|
| **句式特征** | 长短句/句式变化/开头方式 | 爱用短句、排比、对仗 |
| **用词习惯** | 词汇偏好/口头禅/专属用语 | 频繁使用「赫然」「霎时」|
| **节奏把控** | 章节节奏/场景节奏/情绪节奏 | 快节奏战斗/慢节奏日常 |
| **描写密度** | 描写与叙述的比例 | 修仙文高密度/小白文低密度 |
| **情感表达** | 直接抒情/间接表达/情绪词 | 大量内心独白/通过动作暗示 |

## 技巧提炼清单

### 可复用的技巧

```
┌─────────────────────────────────────────────────────────┐
│              技巧提炼清单                                │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  【钩子技巧】                                            │
│  1. 悬念式：「他不知道的是...」（引发好奇）             │
│  2. 冲突式：「就在这时，他来了」（制造紧张）             │
│  3. 反转式：「原来一切都是他策划的」（认知颠覆）        │
│  4. 期待式：「三天后的比试...」（预示精彩）             │
│                                                          │
│  【装逼打脸技巧】                                        │
│  1. 递进式：嘲讽→升级嘲讽→打脸（三步走）               │
│  2. 第三方打脸：无需主角出手，别人替打                   │
│  3. 信息差打脸：展示对方不知道的信息                    │
│  4. 身份曝光打脸：亮出真实身份                          │
│                                                          │
│  【情绪渲染技巧】                                        │
│  1. 递进渲染：悲伤→更悲伤→爆发                         │
│  2. 对比渲染：之前vs现在                                 │
│  3. 环境映衬：悲景写悲情                                 │
│  4. 动作暗示：不用「悲伤」，用动作体现                  │
│                                                          │
│  【对话技巧】                                            │
│  1. 潜台词：说的话和意思不一样                           │
│  2. 打断式对话：话说到一半被打断                         │
│  3. 省略式对话：不说全，留悬念                           │
│  4. 动作配合：「XXX」他XX（动作为主）                   │
│                                                          │
│  【节奏技巧】                                            │
│  1. 欲扬先抑：先压抑再释放                              │
│  2. 持续升温：小爽→中爽→大爽                            │
│  3. 张弛有度：紧张战斗→轻松日常                         │
│  4. 悬念接力：一个悬念接一个                             │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 模板提炼

**开篇模板**：
```
「[背景/状态]，却遭遇[冲突/危机]」
「正当[期待]时，[意外发生]」
「[震撼信息]，主角[反应/决定]」
```

**章末模板**：
```
「就在他以为[预期]的时候，[转折]」
「[新角色/新事件]的出现，让局势变得[描述]」
「他不知道的是，[隐藏信息]」
```

**打脸模板**：
```
「[嘲讽者]：[嘲讽内容]」
「[旁观者]：[期待看戏]」
「主角：[淡定/冷笑]，[行动/反击]」
「结果：[嘲讽者反应]，[震惊/后悔]」
```

## 仿写练习指南

### 仿写步骤

```
┌─────────────────────────────────────────────────────────┐
│              仿写练习四步法                              │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  【第一步】原文精读                                      │
│  · 通读目标段落（300-500字）                           │
│  · 标注：钩子位置/句式特点/用词偏好/节奏把控           │
│                                                          │
│  【第二步】特征提取                                      │
│  · 列出该段的核心特征（3-5个）                         │
│  · 总结：用了什么技巧达到什么效果                       │
│                                                          │
│  【第三步】换题仿写                                      │
│  · 保留结构/技巧，换一个新主题                          │
│  · 例如：原文是「装逼打脸」，仿写成「告白被拒」        │
│                                                          │
│  【第四步】对比优化                                      │
│  · 对比原文和仿写稿的差异                               │
│  · 分析：哪个技巧没掌握/哪个用过头了                    │
│  · 修改仿写稿                                            │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 仿写示例

**【原文】**（装逼打脸场景）
```
「就凭你？」王浩嗤笑一声，「一个废物，也配和我争？」

周围的人也跟着笑起来，有人甚至开始起哄。

叶尘没有说话，只是从口袋里掏出一张名片，轻轻放在桌上。

「这是我的名片。」

王浩低头一看，脸色瞬间变得惨白——

那是城主府特邀顾问的身份证明。

「你...你是...」

「现在，还觉得我配不上吗？」叶尘端起茶杯，淡淡道。
```

**【仿写练习1】换题（学霸打脸）**
```
「就这成绩也敢和我比？」李明把成绩单拍在桌上，满脸嘲讽。

张伟低着头不说话，周围的同学窃窃私语。

王芳默默从书包里拿出一份文件——全国数学竞赛一等奖证书。

「这是我的成绩单。」

李明看了一眼，脸上的嘲讽凝固了——满分750，她考了748。

「...」

「还比吗？」王芳收拾好书包，头也不回地走了。
```

**【技巧分析】**
```
结构分析：
1. 嘲讽者嚣张（建立冲突）
2. 主角沉默/低调（蓄势）
3. 亮出底牌（反转准备）
4. 结果揭露（打脸高潮）
5. 主角淡定收尾（完成装逼）

可复用点：
· 句式：「就凭你？」「现在，还觉得...吗？」
· 节奏：嘲讽(短) → 沉默(短) → 底牌(短) → 结果(长) → 收尾(短)
· 情绪：外放嘲讽 → 内敛反击
```

## 推荐学习书单

### 按风格分类

**快节奏爽文类**：
- 《大王饶命》- 吐槽流、幽默感
- 《飞刀在我是全球首富》- 节奏把控
- 《全职艺术家》- 轻松日常+装逼

**慢热精品类**：
- 《庆余年》- 权谋、人情世故
- 《择天记》- 世界观、人物弧线
- 《雪中悍刀行》- 文笔、武侠韵味

**悬疑推理类**：
- 《诡秘之主》- 氛围营造、伏笔
- 《我有一座恐怖屋》- 恐怖节奏
- 《十方武圣》- 悬疑与战斗

**情感细腻类**：
- 《最好的我们》- 青春共鸣
- 《何以笙箫默》- 情感表达
- 《你是我的荣耀》- 甜文节奏
