## 核心设定

### 核心设定7要素

| 要素 | 说明 | 示例 |
|------|------|------|
| **世界本质** | 这个世界的根本规则是什么 | 修仙世界/科技世界/魔法世界 |
| **时间背景** | 故事发生的时间节点 | 现代/古代/未来/架空纪年 |
| **空间范围** | 故事涉及的主要地域 | 城市/国家/大陆/多界/宇宙 |
| **社会形态** | 这个世界的基本组织形式 | 宗门制/王朝制/联邦制/部落制 |
| **核心冲突** | 世界层面的主要矛盾 | 正邪对立/种族战争/资源争夺 |
| **特殊元素** | 这个世界的独特之处 | 灵气/异能/魔法/系统/科技 |
| **禁忌/铁则** | 绝对不能违反的规则 | 无法复活/修炼有代价/异能有副作用 |

### 快速构建5问

```
1. 这是一个什么类型的世界？（都市/玄幻/科幻/武侠...）
2. 这个世界最大的特点是什么？（灵气复苏/异能觉醒/魔法体系...）
3. 社会如何组织？（宗门/王朝/联邦/学院...）
4. 力量如何划分？（等级/境界/职阶...）
5. 核心冲突是什么？（生存/复仇/登顶/守护...）
```

## 力量体系

### 力量体系类型

| 类型 | 特点 | 示例 |
|------|------|------|
| **等级制** | 明确的等级划分 | 斗者/斗师/大斗师... |
| **境界制** | 境界层层突破 | 炼气/筑基/金丹/元婴... |
| **职阶制** | 不同职业路线 | 战士/法师/刺客/牧师... |
| **天赋制** | 天赋决定上限 | 血脉/神体/特殊体质... |
| **科技制** | 科技发展路线 | 机械/机甲/基因/能量武器... |
| **混合制** | 多种力量并存 | 斗气+魔法/异能+科技... |

### 历史事件设计要点

| 要点 | 说明 |
|------|------|
| **时间节点** | 距今多久，对现在有何影响 |
| **参与方** | 谁参与了这一事件 |
| **核心冲突** | 事件的核心矛盾是什么 |
| **结果** | 对世界格局产生了什么影响 |
| **遗存** | 留下了什么遗迹/传承/恩怨 |
| **争议** | 不同势力对这一事件有何不同解读 |


### 限制与平衡

| 限制类型 | 示例 | 作用 |
|----------|------|------|
| **资源限制** | 灵石/丹药有限 | 控制实力增长速度 |
| **资质限制** | 天赋决定上限 | 区分主角与普通人 |
| **代价限制** | 修炼有副作用 | 防止无敌设定 |
| **数量限制** | 强者稀少 | 保持紧张感 |
| **环境限制** | 特殊地点/时间 | 制造冲突机会 |
