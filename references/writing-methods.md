## 叙事结构

### 三幕式结构

| 幕 | 内容 | 比例 |
|----|------|------|
| 第一幕 | 建置（设定背景、引入冲突） | 25% |
| 第二幕 | 对抗（主角行动、遭遇障碍） | 50% |
| 第三幕 | 解决（高潮对决、结局） | 25% |

### 起承转合

```
起：引入情境
承：发展情节
转：转折/高潮
合：收束/结局
```

### 英雄之旅（12阶段）

1. 普通世界
2. 冒险召唤
3. 拒绝召唤
4. 遇见导师
5. 跨越门槛
6. 考验/盟友/敌人
7. 接近洞穴
8. 严峻考验
9. 获得奖赏
10. 返程
11. 重生
12. 带回灵药

## 人物塑造

### 角色设定要素

| 要素 | 说明 | 示例 |
|------|------|------|
| 外貌 | 视觉特征 | 高、瘦、眉间有疤 |
| 性格 | 核心特质 | 固执、外冷内热 |
| 背景 | 过往经历 | 孤儿院长大 |
| 动机 | 核心欲望 | 为父报仇 |
| 缺陷 | 性格弱点 | 过度信任他人 |
| 成长 | 弧线变化 | 从懦弱到勇敢 |

### 人物关系矩阵

- 主角 ↔ 盟友（信任、支持）
- 主角 ↔ 对手（对抗、竞争）
- 主角 ↔ 导师（指导、启发）
- 主角 ↔ 爱人（吸引、冲突）
- 盟友 ↔ 盟友（友情、协作）
- 对手 ↔ 对手（勾结、背叛）

## 场景描写

### 五感描写法

| 感官 | 描写要点 | 示例 |
|------|----------|------|
| 视觉 | 色彩、形状、光影 | 「夕阳把湖面染成金红色」 |
| 听觉 | 远近、质感、节奏 | 「远处传来断断续续的蝉鸣」 |
| 嗅觉 | 气味、浓淡、来源 | 「空气中弥漫着雨后泥土的腥气」 |
| 触觉 | 温度、质感、力度 | 「他的手心沁出冷汗」 |
| 味觉 | 滋味、浓淡、变化 | 「舌尖残留着咖啡的苦涩」 |

### 场景描写原则

1. **选择性细节**：选取最有代表性的2-3个细节
2. **情绪基调**：细节服务于整体氛围
3. **动静结合**：静态描写穿插动态变化
4. **留白节制**：避免过度堆砌

## 对话设计

### 对话功能

- **推进情节**：揭示信息、触发事件
- **展现性格**：语言习惯反映角色特质
- **制造冲突**：意见分歧、误会争执
- **潜台词**：言外之意、人物心理

### 对话自然度检查

| 检查项 | AI味表现 | 人类化改进 |
|--------|----------|------------|
| 开头 | 「我认为...」 | 「我觉得吧...」 |
| 回应 | 直接肯定/否定 | 先犹豫/找借口 |
| 信息量 | 一次性说完 | 分多次、留悬念 |
| 情绪 | 直接表达 | 通过动作/停顿暗示 |

### 对话示例对比

**AI味**：
```
「你好，我是小明。」他说道。
「你好，小明。」她回应道。
「很高兴认识你。」他说道。
「我也是。」她微笑道。
```

**人类化**：
```
他伸出手，「小明。」
她握住，掌心有点凉，「你好。」
「听说你——」他顿了顿，「算了，没什么。」
```

## 类型写作要点

### 都市言情

- 情感线推进要自然
- 冲突设计要合理
- 配角戏份适度
- 避免狗血套路

### 玄幻修仙

- 等级体系清晰
- 升级节奏把控
- 地图扩展有序
- 战斗描写精彩

### 悬疑推理

- 线索埋藏技巧
- 误导设计
- 反转时机
- 逻辑自洽

### 科幻

- 世界观自洽
- 科技设定可信
- 人文思考深度
- 情节与设定平衡
